3-7 Charts

This lab is about a JavaScript charting library called Chart.js. You can use charts to visualize complicated data. In addition, this library also demonstrates a common design pattern in JavaScript: using JSON data to provide complex options.

Basic operation

We call the constructor of class Chart to create a chart. The constructor accepts two parameters: the first is a <canvas> element, and the second is a JSON data object (a JavaScript object) which has at least three properties: chart type, data to be plotted, and options. Try the demo p371.html.

<!-- a canvas where we'll draw a bar chart -->
<canvas id="myChart" width="400" height="400"></canvas>

<!-- load the Chart.js library -->
<script src="Chart.js"></script>
<script>
// search for the canvas element, and
// pass as 1st parameter to the constructor
var canvas = $("#myChart");
// the 2nd parameter is JSON data
var chart = new Chart(canvas, {
    type: 'bar',
    data: data,
    options: options
  });
</script>

First, chart is an object. You can check its structure in the browser. You can even change the data and update the chart.

dir(chart); // check object structure
// change the data in the 1st dataset
chart.data.datasets[0].data = [ 1, 2, 3, 4, 5, 6, 5, 4, 3, 2, 1];
chart.update();

Next, let’s examine the data and options properties. They are JSON data object, and can get quite complicated. So let’s start with a simpler version. Below is a simplified version of p371.html.

var chart = new Chart(canvas, {
  type: 'bar',
  data: {
    labels: [ 'A', 'A-', 'B+', 'B', 'B-', 'C+', 'C', 'C-', 'D+', 'D', 'F' ],
    datasets: [{
      label: '#students',
      data: [ 3, 3, 5, 6, 4, 4, 7, 4, 2, 5, 5 ],
    }]
  },
  options: { }
});

Now, check the online reference for how p371.html specifies the color of the plot.

More datasets and options

Notice that the property datasets has array value, because the chart can display more than one dataset. See p372.html for an example of two datasets. This example also shows how the options object groups various settings. Please refer to online reference for detail.
