3-2 Select and change

This lab shows how to select elements in the document tree using the $() jQuery function. You may then invoke methods on the resultant collection of elements.

Select by CSS selectors

This part uses the example file p302.html.

The jQuery library provides a powerful function to select elements from the document tree using CSS selectors. This function is sometimes known as the jQuery function $(). When provided with a CSS selectors, it returns a collection of HTML elements that are matched by the CSS selector.

Some examples:

  • $('#mesg') selects the element with id = mesg
  • $('.big') selects all elements that belong to class ‘big’
  • $('a') selects all elements of type a (i.e. all hyperlinks)
  • $('a[href^="http"]') selects all <a> elements with ‘href’ attribute that starts with “http”
  • $('img[alt="tiger"]') selects all <img> elements with ‘alt’ attribute that equals to “tiger”
  • $('img[src$="png"]') selects all <img> elements with ‘src’ attribute that ends with “png”
  • $('img[src*="tiger"]') selects all <img> elements with ‘src’ attribute that contains “tiger”

Change content and attributes

The result of $(selector) is a JavaScript object with many built-in methods. This is often called a jQuery collection. You typically use the methods to manipulate the HTML elements in a collection. The following retrieve or change the content of the selected elements.

  • $(selector).html() returns the HTML content of the first element in the collection
  • $(selector).html(value) sets the content of all the elements in the collection
  • $(selector).text() concatenates and returns the text content of all the elements in the collection
  • $(selector).text(value) sets the content of all the elements in the collection, and escapes the value with HTML entities, if required.

To access the attributes (e.g. src of <img>), use the following:

  • $(selector).attr(attrName) retrieves the attribute value of the named attribute of the first element in the collection
  • $(selector).attr(attrName, value) sets the attribute value of the named attribute of all elements in the collection

Select by structure

When the HTML elements you want to select are embedded in other elements, you can use descendant selector, child selector and pseudo-class like :nth-child(n). Test the following collection in the example file p304.html.

  • $('table#marks td.fail') selects some table cells <td> inside a <table> with id = ‘marks’
  • $('ol > li:first-child') selects the first child of <ol>
  • $('ol li:first-child') selects all <li> that is itself a first child. The element must be inside <ol>

If you want to do your own processing on the elements in a jQuery collection, iterate the collection with a for .. of loop. Notice that each element retrieved by for .. of loop is a DOM element object. To use jQuery methods, convert the object into a jQuery collection with $(). Test the following in p304.html.

// this example adds 1 mark to exam of everybody!
// first, select all cells in the table body
var $cells = $('table#marks tbody td:nth-child(3)');
// for each HTML element in the jQuery collection ...
for (var c of $cells) {
  // use $() to make the DOM node into a jQuery collection
  // then, we can use .text() to read content
  var mark = $(c).text();
  mark = parseInt(mark);
  $(c).text(mark+1);
}

You can also use the $().each() method (see online reference).

// same effect as above, rewritten using $().each()
// first, select all cells in the table body
var $cells = $('table#marks tbody td:nth-child(3)');
// for each HTML element in the jQuery collection ...
$cells.each( (idx, c) => {
  // use $() to make the DOM node into a jQuery collection
  // then, we can use .text() to read content
  var mark = $(c).text();
  mark = parseInt(mark);
  $(c).text(mark+1);
});

Sometimes, you want to search for elements under a subtree of the DOM tree. You can use $().find(selector) to search for elements under the current matched set. For example, $('div#a').find('img') will return all <img> that are inside the <div> with id = a. Notice that $('div#a').find('img') gives the same jQuery collection as $('div#a img').

  • $().find(selector) selects descendants under each element in the jQuery collection. The descendants must match the selector, if specified.
  • $().children(selector) selects direct children under each element in the jQuery collection. The children must match the selector, if specified.
  • Similarly, you can find the parents $().parent() or siblings $().next(), $().prev(), $().siblings() of elements in a jQuery collection. See online reference for detail.

Consider this example. We want to calculate the final mark for the students in table#marks. Final mark = 40% x test mark + 60% x exam mark. Run this example in p304.html.

// for each row in the table body
for (var row of $('#marks tbody tr')) {
  // get the test mark from the 2nd cell
  var testmark = $(row).find('td:nth-child(2)').text();
  testmark = parseFloat(testmark);
  // get the exam mark from the 3rd cell
  var exammark = $(row).find('td:nth-child(3)').text();
  exammark = parseFloat(exammark);
  // calculate final mark and show it in the 4th cell
  var final = testmark * 0.4 + exammark * 0.6;
  $(row).find('td:nth-child(4)').text(Math.round(final));  
}

Select by position

You can select elements in a jQuery collection by position.

  • $(selector).eq(n) selects a single element at position n from the jQuery collection. n starts from 0
  • $(selector).length returns the number of elements
  • $(selector).slice(m,n) selects a range of elements

Change CSS styles

There are two ways to change the CSS styles of elements. The first method is to add inline style with $().css(). Try the examples in p307.html.

  • $(selector).css(property, value) sets the CSS property to the specified value for all elements in the collection

When you need to set many CSS properties, a more organized method is to write the style change as style rules in CSS style sheet. Assign the style rules to a class.

.center {
  position: fixed; top: 30%; left: 30%;
  width: 300px !important; height: 300px !important;
  background-color: lightblue; border: 1px solid gray;
  border-radius: 30px; padding: 10px;
}

Then, in JavaScript, you can add the class to the element in runtime with $().addClass(). This approach also makes it easy to ‘undo’ the style change by removing the class from the element with $().removeClass(). Try the examples in p308.html.

  • $(selector).addClass(className) adds the specified class to the elements in the collection
  • $(selector).removeClass(className) removes the specified class from the elements in the collection

Next, define a style rule for the table content and apply the rule to the table element in JavaScript. You may use the following sample. When you’re done, your answer should look like this.

$('table#z').addClass('mystyle')
