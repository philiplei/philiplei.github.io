Try each of the following.

  • $('#mesg').css('color', 'white').css('background-color', 'red')
  • $('#mesg').css(...) - change font related properties (e.g. font-family, font-size). (online reference)
  • $('div.dialog').css('width', '100px').css('border', '1px solid green')
  • $('#z tbody tr:nth-child(even)').css('background-color', 'lightblue')
  • $('#z img').css('transform', 'rotate(45deg)') - rotate all images by 45deg (online reference)
  • $('#z img[src*="tiger"]').css('width', '120px') - change the width of the image with a image file name that contains "tiger"
  • $('#z img').eq(4).css('transform', 'scale(3,3) translate(30px,0)') - resize by 2 times
  • $('#z img').slice(2,4).css('transform', 'translate(200px,0) rotate(90deg)')

Reload this page before trying the following.

  • $('#z img').eq(0).addClass('center') - add a class to the element. This HTML code includes some style rules for the class center
  • $('#z img').eq(0).removeClass('center') - remove the class from the element
