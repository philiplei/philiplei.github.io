Create and insert:

  • Create a dragon image with var d = $('<img>').attr('id', 'dragon').attr('src', 'ani/dragon.png')
  • Note: this image is not shown because it is not attached to the current document tree yet
  • Append this image to the start of div#zone1 with $('#zone1').prepend(d);
  • Create a pig image with var p = $('<img>').attr('id', 'pig').attr('src', 'ani/pig.png')
  • Insert this image before the rooster image with $('#rooster').before(p);
  • Repeat the experiment with .append() and .after() ...

Move existing elements

  • Note: if the source element is already attached to document tree, it is removed from the original location and moved to the target.
  • Select the ox image (id='ox') and move it to the front of the tiger image.
    var e = $('#ox'); $('#tiger').before(e);, or in 1 statement:
    $('#tiger').before($('#ox'))
  • Next, move the ox image behind the last image (rooster)
  • Move all images in div#zone1 to div#zone2 with $('#zone2').append($('#zone1 img'))

Clone existing elements

  • Make a clone of the first image, and append it to #zone3: var c = $('img').eq(0).clone(); $('#zone3').append(c);

When there are than 1 target, The source content are cloned on demand.

  • $('#zone2,#zone3').append($('#ox')) - move the ox image to two targets
  • $('img').after($('<img>').attr('src', 'ani/dragon.png')) - insert a dragon image after every image
