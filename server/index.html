4-6 Database access

This lab demonstrates how to use a file-based database sqlite. SQLite SQLite is a software library that implements a self-contained, serverless, zero-configuration, transactional SQL database engine (from https://www.sqlite.org/). The whole database is stored in a file, and an application can open this database directly without connecting to any server. A popular module to access SQLite in Node is sqlite3. Install this module and update package.json with the command npm install sqlite3 --save. »
