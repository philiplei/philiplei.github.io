4-5 Templates and file modules

This lab demonstrates how to use templates in Express app. In addition, it shows how to break a JavaScript program into modules by using ‘file modules’.

Template engine

In previous lab, we embedded HTML template inside JavaScript code to render data. This approach becomes impractical when the HTML code or the data become more complicated. A better approach is to use some template system to render data in the server side.

We studied how to use the Handlebars template system in Lab 3-6 for client-side. It is also possible to use the template system in an Express app with a suitable template engine. A popular module is hbs. Install in with the command npm install hbs --save or npm install with suitable entry in package.json. Then, in the Express app, load the engine with the following line:

// curiously, you don't load with ..
// var hbs = require('hbs');

// set the template engine to handlebars
app.set('view engine', 'hbs');

By default, Express searches for templates in the folder /views and assume the file extension .hbs.

app.get('/year/:y', (req, res) => {
  var year = parseInt(req.params.y);
  var coursesInTheYear = timetab.courses.filter((c)=>c.year==year);
  var context = {
    year: year,
    acadYear: timetab.acadYear,
    semester: timetab.semester,
    courses: coursesInTheYear
  };
  // use the template '/views/list.hbs' to render the context object
  res.render('list', context);
});

File modules

In addition to built-in modules and NPM modules, there is a third kind in Node.js. A file module is a *.js file that exports some functions or objects. The module exports functions by adding them to the special object exports.

// module dice.js

// private function
function randomInteger(min,max) {
  return Math.floor(Math.random()*(max-min+1)) + min;
}

// functions to be exported
function rollDice() { return randomInteger(1,6); }

function rollDiceN(n) {
  var ans = [];
  for (var k=0; k<n; k++) { ans.push(randomInteger(1,6)); }
  return ans;
}

function drawOne(items) {
  // assume item is a non-empty array
  var k = randomInteger(0, items.length-1);
  return items[k];
}

// export functions, classes, variables by attaching them
// as properties to the special object 'exports'
exports.rollDice = rollDice;
exports.rollDiceN = rollDiceN;
exports.drawOne = drawOne;

The following ‘main program’ imports the file module and uses the exported funtions.

// main.js

// import a file module by specifiying a file name with path
// here, the module is in the same folder as main.js
var d = require('./dice.js');

// now the exported functions are accessible as
//   d.rollDice, d.rollDiceN, and d.drawOne
var letters = 'ABCDEFG'.split('');
console.log(`Roll dice 10 times: ${d.rollDiceN(10)}`);
console.log(`Draw a letter from 'ABCDEFG': ${d.drawOne(letters)}`);
