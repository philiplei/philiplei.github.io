2-2 Arrays and functions

This lab goes through the basics of the JavaScript arrays and functions.

Array

An array is a sequence of values that may have different types. Arrays can change size in run-time. The length property refers to the number of elements inside the array. Use the operator [] to locate an element at the given index.

var empty = [ ];
var numbers = [ 'zero', 'one', 'two', 'three', 'four' ] ;
empty[1] // undefined
numbers[1] // 'one'
empty.length // 0
numbers.length // 5

A common method to iterate an array is to use for loop.

var P = [ 2,3,5,7,11 ] ;  // small primes
for (var i=0; i<P.length; i++) {
  console.log(P[i]);
}

ES6 has a new control structure for of to iterate arrays and other iterable objects.

var P = [ 2,3,5,7,11 ] ;  // small primes
for (var x of P) {
  console.log(x);
}

Basic array methods

Arrays are objects in JavaScript and have some built-in methods. See online reference for a list of all methods. The following shows some common methods.

var A = [2,4];
A.push(6)  // A becomes [2,4,6]
A.pop(); // returns 6. A becomes [2,4]
A.unshift(0); // A becomes [0,2,4]
A.shift(); // returns 0

var b = [202,175,3,3];
var s = b.join('.'); // "202.175.3.3"

Arrays are useful to keep some values in order. The following example generates prime numbers less than 200. Please refer to online help for the detail usage of break and continue.

var P = [2, 3, 5, 7];
// give a label to the first for loop
loop1:
for (var k=9; k<200; k+=2) {
  loop2:
  for (var n of P) {
    // if k is divisible by a prime in P, continue onto k+2
    if (k % n == 0) continue loop1;
  }
  // after the above for loop, k is verified as prime
  P.push(k);
}
console.log(P);

Functions

There are two ways to define functions: function keyword and arrow function. This lab covers the first. We use a function to group some statements that can be reused.

// draw a right angle triangle of height n
function draw (n) {
  var out = "";
  for (var y=0; y<n; y++) {
    var s = "";
    for (var i=0; i<n-y-1; i++) s += ". ";
    for (var i=0; i<=y; i++) s += "* ";
    out += s+"\n";
  }
  console.log(out);
}

draw(5); // call the function to draw a triangle of height 5
draw(10); // draw again

Functions can return values to the calling code with return. The return statement also ends the execution of the functions immediately.

// this function compares the two parameters a and b, and returns the bigger one
function bigger (a,b) {
  if (a>b) {
    return a;
  } else {
    return b;
  }
}
console.log('The bigger one is ', bigger(2,3));

// another way to write
function bigger2 (a,b) {
  if (a>b) { return a; }
  return b;
}

// yet another way to write
function bigger3 (a,b) { return a>b ? a : b; }

The next example defines a function that find the largest numbers in an array, assuming that the array has at least 1 element.

function max (A) {
  var x = A[0]; // assume the 0th element is the largest
  // compare x with each remaining element
  for (var k=1; k<A.length; k++) {
    if (A[k]>x) x = A[k];
  }
  return x;
}

max([2,3,5,8,7,1,4,6]); // return 8

In JavaScript, variables have function scope. That means that variables defined within a function are only visible inside the function. They are sometimes called local variables. Function parameters also work like local variables. A new copy of These variables is created when the function executes, and removed when no code has access to them anymore. For simple case like the above example, the local variables are removed when the function returns.

On the other hand, variables defined outside any functions are global variables, and are visible inside any functions.

  1. Write a function sum(A) to calculate the sum of elements in an array A. Use the following code to test your function.

    var x = [1,2,3,4,5];
var ans = sum(x);
console.log('Correct? ', ans==15);

  2. Write a function odd(n) to generate an array of odd numbers less than or equal to n. Test your function with the following code.

    var a = odd(1);
console.log(a); // should print [1]
console.log(odd(9)); // should print [1,3,5,7,9]
console.log(sum(odd(10*2-1))); // should print 100

  3. Write a function search(A, n) to search for an number n in the array A. Returns the position of the element if it is found. If it is not found, return -1. Test your function with the following code.

    var x = [6, 8, 3, -2];
console.log(search(x, 3)); // should print 2;
console.log(search(x, 1)); // should print -1;
